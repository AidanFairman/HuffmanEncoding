To use the program, run from the console

Usage to compress any document:
"HuffmanEncoding.exe [fileToCompress.txt] -c"

Usage to decompress any already compressed document (compressed by this program):
"HuffmanEncoding.exe [comp-fileToCompress.txt] -d"

War and Peace is provided as a testing document if you need somthing to compress, or if you want to read it.
War and Peace provided by the Project Gutenberg library
